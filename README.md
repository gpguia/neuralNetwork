# Neural Networks

This is a simple version of a neural network.

## Getting Started

To compile it just type:
```
make
```

## Running the tests

To run just type:

```
./neuralNet
```
It will ask you to enter some values:

```
Please type the number of input nodes:
Please type the number of hidden nodes:
Please type the number of repetitions:
Please type the Learning Rate: (here you must type a number from 0.05 to 0.5).
```

After that the program will generate and run the code.
